Сегодня мы открыли позиционные скидки в Биллинге по двум продуктам: КЭ и EDI. Позиционные скидки – это скидки на отдельные позиции (тарифы) в счете.

По КЭ скидки доступны всем, ограничения размера скидок являются общими для всех. Для КЭ режим скидок может быть выбран пользователем: либо по прежней схеме скидка на весь счет, либо позиционные скидки.

По EDI скидки доступны только супервизорам:
